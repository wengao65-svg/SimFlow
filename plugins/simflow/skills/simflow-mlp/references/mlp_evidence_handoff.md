# MLP Evidence Handoff

Handoff packages should include:

- Goal, material/domain, MLP family, toolchain, and current iteration id.
- Evidence roles: dataset manifest, labeling manifest, training run, model artifact, validation report, long-MD or smoke-MD evidence, anomaly report, active-learning decision, and approval status.
- Missing evidence and claims blocked by those gaps.
- Next actions and whether user approval is required.

Keep raw files and generated summaries linked through artifact metadata.
